My Work

Note Manager

Notemanager is an app that you can register with your email and start taking and managing your daily notes. You can manage the notes by editing them, also you can filter the notes based on date, note text etc. App has features of register and login. It uses JWT authentication so it's fully client side. It uses Node and Express to serve the app, and MongoDB to Register User's, Login, Adding Deleting and Updating Notes.

Technologies

  • Vue.js, Vuex
  • Bootstrap, SASS
  • Node.js, Express.js
  • MongoDB

Diamond Chat

DiamondChat is an chat app that aims to connect people same with their music taste. This is demo for now. People can sign in with their Facebook account and currently there is just random chat functionality. There will be a login with Spotify functionality and I will try to get data from Spotify and match the people according to their music taste.

Technologies

  • Vue.js, Vuex
  • Bootstrap, SASS
  • Node.js, Express.js, Socket.io, TypeScript
  • MongoDB

Twitter Analyzer

Twitter Analyzer is an app where you give a keyword and start analyzing tweets accordingly in real time. It gives you a realtime dashboard to track Tweets info, also it gives you a world map representation to show you a Tweet's location in real time.

Technologies

  • React.js
  • Node.js
  • CSS3
  • Spring Boot
  • Spring Websocket

Target Sum Game

Target Sum Game is a game that you need to reach the target sum number by clicking buttons. When you click buttons, the value of button added to current sum value, when it becomes equal to target sum number, you pass a stage, if the value is bigger than the current value game over :)

Technologies

  • React.js, Redux
  • Node.js
  • Sass
